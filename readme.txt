OBJECT OF THE GAME
Each participant attempts to beat
the dealer by getting a count as 
close to 21 as possible, without going over 21.

CARD VALUES/SCORING
It is up to each individual player if an ace is
worth 1 or 11. Face cards are 10 and any other 
card is its pip value.

How to run the game(if you want to make changes)
1.Have at least C++14 installed, C++ 17(recommended)
2.Clone repository
3.Open your favorite cmd prompt, (bash,powershell,terminal etc)
4.Run the make file
5.Enjoy the game!

Quick way to play the game
1. Clone Repository
2. Double click on main.exe file
3. Enjoy the game!
